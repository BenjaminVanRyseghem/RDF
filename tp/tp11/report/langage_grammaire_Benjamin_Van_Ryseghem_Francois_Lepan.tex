\section{Arbre de dérivation pour une grammaire}
\subsection{A la main : la grammaire G}
\begin{verbatimtab}
  - Alphabet A = {a,b,c}
  - Axiome = S
  - Non-terminaux = {A,B}
  - Règles de production P =
	S --> cAb
	A --> aBa
	B --> aBa
	B --> cb
\end{verbatimtab}
\subsubsection{De quel type est cette grammaire?}

C'est une grammaire algébrique car elle est de la forme $R_i:T\rightarrow x$ où x est terminal ou non

\subsection{Montrez que cette grammaire génère le langage L(G) = \{$c a^n cb a^n b$ | $n \geq 1$\}}

On peut voir sur la Fig.~\ref{grammar_abc} que à chaque fois on rajoute autant de a de chaque coté du cb centrale (cas d'arrêt).
